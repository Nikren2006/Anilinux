# 🎬 AniLinux

Desktop-приложение для просмотра аниме с интеграцией **Shikimori API** и парсером **Animevost**.

[![Release](https://img.shields.io/badge/Release-v1.0.0-blue.svg)](https://github.com/YOUR_USERNAME/anilinux/releases)
[![Python](https://img.shields.io/badge/Python-3.10+-blue.svg)](https://www.python.org/downloads/)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)
[![Platform](https://img.shields.io/badge/Platform-Linux-x86_64-lightgrey.svg)](https://www.linux.org/)

> **AniLinux** — ваш бесплатный десктопный клиент для просмотра аниме с русской озвучкой. Ищите аниме через Shikimori, смотрите через Animevost, отслеживайте прогресс и собирайте коллекцию избранного.

---

## ✨ Возможности

- 🔍 **Поиск аниме** через Shikimori API с рейтингами, описаниями и жанрами
- 🎬 **Просмотр аниме** — парсер Animevost с получением прямых ссылок на видео
- 📺 **Видеоплеер mpv** — воспроизведение в отдельном окне с управлением
- ⭐ **Избранное** — список аниме с отслеживанием статуса
- 📜 **История просмотров** — автоматическое сохранение просмотренных серий
- 📊 **Статистика** — время просмотра, количество эпизодов, загрузки
- 📥 **Загрузки** — скачивание эпизодов для оффлайн-просмотра
- 🗂️ **Кэширование** — изображений и данных
- 🎨 **Адаптивный дизайн** — карточки перестраиваются при изменении размера окна

## 🚀 Установка

### Требования

| Зависимость | Описание |
|-------------|----------|
| **mpv** | Видеоплеер для воспроизведения аниме |

### Arch Linux / Manjaro

```bash
sudo pacman -S mpv
```

### Ubuntu / Debian

```bash
sudo apt install mpv
```

### Fedora

```bash
sudo dnf install mpv
```

### Запуск

1. Скачайте последнюю версию из [Releases](https://github.com/NikRen2006/Anilinux/releases)
2. Запустите:

```bash
./AniLinux
```

Или через скрипт:

```bash
./run.sh
```

> **💡 Совет:** При первом запуске автоматически создаётся:
> - **Иконка в меню приложений** — ищите «AniLinux»
> - **CLI-команда `anilinux`** — можно запускать из любой директории:
>   ```bash
>   anilinux
>   ```
> 
> Если `~/.local/bin` не в PATH, добавьте в `~/.bashrc` или `~/.zshrc`:
> ```bash
> export PATH="$HOME/.local/bin:$PATH"
> ```

## ⚙️ Настройки

Настройки доступны в разделе **«Настройки»** боковой панели:

- **Shikimori API Token** — для авторизации в Shikimori
- **Директория загрузок** — путь для сохранения загруженных файлов
- **Кэш** — очистка кэша изображений

## ⚠️ Дисклеймер

AniLinux не хранит и не распространяет видеоконтент. Приложение является клиентом для доступа к publicly доступным данным через Shikimori API и Animevost. Все права на аниме принадлежат их правообладателям.
